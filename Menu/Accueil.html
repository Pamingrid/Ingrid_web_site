Création de compte

L'accès à l’application BIAPAY s'effectue en entrant l'URL de BIAPAY dans la fenêtre d’un navigateur Web. Depuis la page d'accueil du site, on peut s’inscrire, se connecter et accéder au contenu et aux fonctions de l’application.
Dans BIAPAY, il existe deux type de compte à savoir : Le compte privé et compte professionnel. Pour créer un compte dans BIAPAY, procéder comme suit :

  1. Visiter le site web de BIAPAY
  2. Cliquer sur le bouton Enregistrer.

Créer un compte utilisateur privé

Pour créer un compte utilisateur privé, procéder comme suit:

  1. Sélectionner comme type de compte "Compte privé" et cliquer sur suivant.
  2. Sélectionner l’indicatif pays, indiquer le numéro de téléphone et cliquer sur suivant.
  3. Renseigner le code de vérification. Un code temporaire est envoyé au numéro de téléphone précédemment saisi pour la vérification de celui-ci.
  4. Remplir le formulaire d’inscription
  5. Une fois le formulaire envoyé, une notification est affichée à l’écran pour notifier que le compte a été créé et vous inviter à consulter la boîte de réception e-mail pour le valider.

Créer un compte utilisateur professionnel

Pour créer un compte utilisateur preofessionnel, procéder comme suit:

  1. Sélectionner comme type de compte Compte privé et cliquer sur suivant.
  2. Sélectionner l’indicatif pays, indiquer le numéro de téléphone et cliquer sur suivant.
  3. Renseigner le code de vérification. Un code temporaire est envoyé au numéro de téléphone précédemment saisi pour la vérification de celui-ci.
  4. Remplir le formulaire d’inscription
  5. Une fois le formulaire envoyé, une notification est affichée à l’écran pour notifier que le compte a été créé et vous inviter à consulter la boîte de réception e-mail pour le valider.

Activer son compte

Pour activer le compte, procéder comme suit :

  1. Ouvrir le courriel de confirmation dans votre boîte de réception, puis cli-quer sur le lien qu’il contient.
  2. Une notification « Compte vérifié » apparait à l’écran, cliquer sur le lien de connexion.
  3. Saisir l’adresse e-mail et le mot de passe.
  4. Cliquer sur le bouton « Connexion ».
  5. Remplir le formulaire Know Your Customer (KYC) et cliquer sur le bouton "Envoyer.
  6. Une fois le formulaire KYC envoyé, la notification sur l’écran change. Elle affiche désormais KYC en cours de vérification.
  7. Le formulaire va alors être soumis à une double vérification par les administrateurs du système. Pour l’utilisateur professionnel, En attente de vérification, l’utilisateur professionnel aura accès en mode lecture.
    Il ne pourra inscrire ou sauvegarder aucune informa-tion sur les autres fonctionnalités de BIAPAY. Le formulaire KYC sera en règle générale vérifié dans un délai maximum de 48 heures. Une fois le formulaire KYC vérifié et validé, son statut change et il est désormais possible d’utiliser le système sans restriction.

    8. Se déconnecter

    Pour se déconnecter à l’application BIAPAY, l’utilisateur connecté doit procéder comme suit :

    1. Cliquer sur l’image de profil utilisateur ; un panneau se déroule et on peut voir le lien déconnexion.
    2. Cliquer sur le lien « Déconnexion ».
    3. L’utilisateur est déconnecté de l’application BIAPAY et la page connexion à l’application apparait.
